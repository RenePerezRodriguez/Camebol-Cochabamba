# Política de Seguridad

## Reportar una vulnerabilidad

Si encuentras una vulnerabilidad de seguridad, **NO abras un issue público**.

Envía un correo a **<info@camebolcochabamba.com>** con:

- Descripción del problema
- Pasos para reproducirlo
- Impacto potencial

## Proceso

1. Confirmamos la recepción del reporte en un máximo de 48 horas.
2. Investigamos y verificamos la vulnerabilidad.
3. Desarrollamos y aplicamos la corrección.
4. Notificamos al reportante una vez resuelto.

## Alcance

Esta política cubre el sitio web [camebolcochabamba.com](https://camebolcochabamba.com) y este repositorio.
