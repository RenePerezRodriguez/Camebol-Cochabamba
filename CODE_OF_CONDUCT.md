# Código de Conducta

Adaptado del [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct.html).

## Compromiso

Nos comprometemos a que la participación en este proyecto sea una experiencia libre de acoso para todos, independientemente de edad, género, orientación sexual, discapacidad, etnicidad, religión o nivel de experiencia.

## Estándares

**Comportamiento esperado:**

- Empatía y respeto en las interacciones
- Aceptar retroalimentación constructiva
- Priorizar el bien del proyecto sobre intereses individuales

**Comportamiento inaceptable:**

- Lenguaje o imágenes ofensivas
- Ataques personales, trolling o acoso
- Publicar información privada de otros sin consentimiento

## Aplicación

Reportar comportamiento inaceptable a **<info@camebolcochabamba.com>**.

Los administradores pueden eliminar, editar o rechazar contribuciones que no cumplan este código.
